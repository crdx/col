# Changelog

## [1.2.0] - 2025-12-05

- Add `Strikethrough` for strikethrough text.
- Add `Reverse` for reversed foreground/background.
- Add `Dim` for dimmed text.

## [1.1.0] - 2024-02-26

- Upgrade minimum Go version to 1.22.
- Add formatting arguments.

## [1.0.3] - 2023-01-19

- Add godoc comments.

## [1.0.2] - 2023-01-18

- Update import paths to use a vanity domain.

## [1.0.1] - 2023-01-17

- Move code out of `src/` as Go does not support such a directory structure. Put it in `internal/` instead.

## [1.0.0] - 2021-12-17

- Initial release.
